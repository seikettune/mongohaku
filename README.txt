A Tiny search demo

Database = MongoDB
MongoDB download: https://www.mongodb.com/download-center#community

Using
NodeJS download: https://nodejs.org/en/download/

A tiny list of modules: bcrypt-nodejs body-parser cookie-parser express express-session mongoose mongoose-search-plugin passport passport-local path
- cd root of app
- npm install --save bcrypt-nodejs body-parser cookie-parser express express-session mongoose mongoose-search-plugin passport passport-local path


Bower:
- .bowerrc redirecting bower components into root of app/public/bower_components
- Using following bower components: bower install --save angular bootstrap angular-ui-router angular-strap angular-cookies
- Installing bower -> npm install -g bower
